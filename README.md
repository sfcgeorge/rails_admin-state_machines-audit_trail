## Installation
Add this line to your application's Gemfile:

```ruby
gem 'rails_admin-state_machines-audit_trail'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install rails_admin-state_machines-audit_trail
```
